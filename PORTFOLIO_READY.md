## 📞 Quick Help

**Site not smooth?**
- Clear cache (Ctrl+Shift+Delete)
- Hard refresh (Ctrl+Shift+R)
- Use Chrome/Firefox (latest version)

**Contact form not sending?**
- Add EmailJS credentials in Contact.tsx
- Test on EmailJS dashboard first

**Colors wrong?**
- Edit `src/components/ui/GlassCard.tsx`
- Change color classes (blue → your color)

**Build fails?**
```bash
rm -rf node_modules && npm install && npm run build
```
